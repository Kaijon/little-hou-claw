# 🎬 Subtitle（繁體中文字幕生成）

> 丟一個 YouTube 連結或影片，小龍蝦幫你生成繁體中文字幕 🦞🈯

## ✨ 這個技能可以做什麼？

- 🎬 為 YouTube 影片生成繁體中文字幕
- 🌐 為遠端影片 URL 生成字幕
- 📁 為本地影片檔案生成字幕
- 📄 輸出標準 SRT 格式（可直接用於大多數播放器）
- 📋 支援 WebVTT 格式輸出
- 💾 可指定儲存到檔案

## 📦 安裝方式

在 Telegram 對話中輸入：

```
/skills
```

輸入後會顯示可安裝的技能列表，選擇此技能即可完成安裝。

## ⚙️ 需要的設定

| 設定項目 | 說明 |
| --- | --- |
| `GEMINI_API_KEY` | Google Gemini API 金鑰，字幕生成功能的核心引擎 |

## 💬 提示詞範例

```text
幫這個 YouTube 影片加上繁體中文字幕 https://www.youtube.com/watch?v=abc123
幫我把這段影片的字幕翻成繁體中文 https://youtu.be/abc123
把這個影片的字幕存成 SRT 檔 ./clips/demo.mp4
幫這部影片生成 WebVTT 格式的中文字幕
```

## 📝 輸出範例

### SRT 格式（預設）

```
1
00:00:01,000 --> 00:00:04,500
歡迎來到今天的節目。

2
00:00:04,800 --> 00:00:08,200
今天我們將探討人工智慧的最新發展。

3
00:00:08,500 --> 00:00:12,000
首先，讓我們來看看最近的突破性研究。
```

### WebVTT 格式（`--format vtt`）

```
WEBVTT

00:00:01.000 --> 00:00:04.500
歡迎來到今天的節目。

00:00:04.800 --> 00:00:08.200
今天我們將探討人工智慧的最新發展。
```

## ⚠️ 注意事項

- 字幕時間精確度取決於 Gemini 模型對影片語音的理解能力
- 過大的本地影片檔案可能超出記憶體限制，建議先壓縮再處理
- 若影片沒有語音（只有背景音樂），則無法生成字幕
- 需要使用者有權限存取的影片可能無法處理
